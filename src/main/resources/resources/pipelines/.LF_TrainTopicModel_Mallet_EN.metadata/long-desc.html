Pipeline LF_TrainTopicModel_Mallet_EN

A pipeline for training a topic model on the filtered tokens for English documents.

This pipeline expects Token annotations with POS tags (in feature "category") in the default annotation set and filters them by token kind, English stop words, and POS tag to create TokenWord annotations in the "LDA" annotation set which is used for training the Mallet topic model.

More information: Pipeline LF_TrainTopicModel_Mallet_EN
